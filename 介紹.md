# Logic Pusher 3D - 專案簡介
> **結合「推箱子」機制與「數位邏輯」原理的 3D 益智解謎遊戲**

**Logic Pusher 3D** 是一款基於 Unity 6 開發的益智遊戲。玩家身處於一個充滿電子訊號的微觀世界，必須透過推動邏輯閘（AND、OR、NOT）與電線方塊，構建出正確的電路路徑，將電源引導至終點以修復系統。

本專案採用 **Contract-First (契約優先)** 開發模式，強調嚴謹的軟體架構與高擴充性的邏輯運算系統。

---

## 🎮 核心玩法 (Core Gameplay)

### 1. 邏輯推箱 (Sokoban Logic)
遊戲保留了經典的推箱子移動限制，但「箱子」變成了具有功能的電路元件。玩家必須在有限的空間內規劃路徑，將元件推至正確位置。

### 2. 方向性電路 (Directional Circuitry)
不同於傳統接水管遊戲，本遊戲中的邏輯閘具有 **嚴格的方向性 (Orientation)**：
* **AND Gate**：需同時滿足後方與側面的輸入訊號。
* **NOT Gate**：僅接受後方輸入並進行反相輸出。
* 玩家不僅要考慮位置，還需思考元件的**朝向**是否正確對接訊號流。

### 3. 即時運算回饋 (Real-time Feedback)
每一次移動或推動結束後，系統會即時重算全場電路。通電的線路會發出高強度的光暈 (Bloom)，讓玩家直觀地看見電流的流向與邏輯錯誤。

---

## ✨ 專案特色 (Key Features)

* **視覺化除錯**：利用 Unity URP Volume 與 Post-Processing 技術，透過材質發光強度 (Emission) 的變化，將抽象的邏輯訊號 (0/1) 轉化為直觀的視覺體驗。
* **穩健的邏輯核心**：內建 **迴圈保護機制 (Loop Protection)**，即使玩家構建出邏輯死循環（如 NOT Gate 頭尾相接），系統也能保持穩定運作而不崩潰。
* **資料驅動關卡**：採用 JSON 格式定義關卡配置與元件初始朝向，支援快速擴充與修改關卡設計。

---

## 🛠 技術架構 (Technical Stack)

本專案旨在展示高品質的程式架構設計能力：

* **Engine**: Unity 6 (6000.0.29f1)
* **Render Pipeline**: Universal Render Pipeline (URP)
* **Architecture**:
    * **MVC 變體**：嚴格區分 Model (純邏輯運算)、View (Unity MonoBehaviour 視覺表現) 與 Controller (輸入控制)。
    * **介面導向 (Interface-Based)**：透過 `IGridSystem` 與 `ICircuitSystem` 降低模組耦合度。
    * **策略模式 (Strategy Pattern)**：封裝各類邏輯閘運算法則，易於新增元件類型。
* **Input**: Unity Input System Package (v1.11+)
